The to do list stores values input in an array in an
object.  The object is todoList, It can:

-log to the console the array through
the .displayToDos method.  

-add a To Do to the array through the .addToDo method.
The change is displayed afterwards.

-change a To Do in the array through the .changeToDo 
method. The change is displayed afterwards.

-delete a To Do in the array through the .deleteToDo
method. The change is displayed afterwards.

-if the To Do is completed you can toggle an x for 
completed with the .completed method.

-toggle the entire list on or off using the .toggleAll
